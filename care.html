// 02. CARE & BREEDING

飼育・繁殖ガイド

温度・湿度・給餌の基本から繁殖の全フェーズ、病気チェッカー、季節カレンダーまで。

// CARE BASICS

飼育の基本

爬虫類を健康に長く飼育するための8つの必須ポイント。

01
🌡️
温度勾配

変温動物は外気温で体温が変わる。ホット（35～45℃）とクール（22～26℃）の勾配を作り自己調節できる環境を。パネルヒーター＋保温球を組み合わせ。

02
💧
湿度管理

砂漠種30～40%、熱帯種70～90%が目安。脱皮不全は湿度不足が原因の多くを占める。ウェットシェルター・霧吹き・自動散水装置を活用。

03
☀️
紫外線 UVB

昼行性のトカゲ・カメはビタミンD3合成にUVB必須。Arcadia 6～12%・Zoo Med ReptiSun 10.0が定評。6～12ヶ月で交換。クル病（MBD）予防の要。

04
🍖
給餌・栄養

肉食種はカルシウムダスティング（週3～5回）必須。コオロギ・デュビア・ピンクマウスが主な活餌。草食種は野草・葉物野菜・専用ペレットを組み合わせ。

05
🏠
シェルター

隠れ家がないと慢性ストレスで死亡することも。ウェットシェルターは局所的高湿度で脱皮を助ける。樹上種にはコルク流木・枝で立体空間を。

06
🧹
清掃・衛生

排泄物はこまめに除去。床材は月1～2回全交換。水入れは毎日洗浄。サルモネラ菌対策として世話後は必ず石鹸手洗い。Virkon等爬虫類用消毒液で月次除菌。

07
🏥
健康管理・獣医

エキゾチック動物専門獣医を事前に探す。食欲不振・脱皮不全・マウスロット・プロラプスは早期受診。月1回の体重測定で異常を早期発見。

08
😴
ブルメーション

熱帯種でも冬に活性が落ちる「ブルメーション」が起こることがある。無理に給餌せず温度・水分管理のみ維持。温帯種の本格冬眠は初心者は室内越冬が安全。

// BREEDING GUIDE

繁殖情報

クーリングからベビーの育成まで5フェーズで解説。

// PHASE 01
クーリング

多くの種で繁殖トリガーとなる低温処理。ホルモン分泌を促すため冬季に温度を段階的に下げる。

  • 秋（10～11月）から徐々に温度を下げる（28℃→22℃程度）
  • 期間は種によって2～8週間が目安
  • 体重十分な健康個体のみ実施（脂肪蓄積15～20%以上）
  • クーリング中も水は切らさない
  • 給餌は停止または大幅減少
⚠ 病気・痩せ個体のクーリングは死亡リスクあり。必ず健康確認を。
// PHASE 02
ペアリング・交尾

温度回復後にオスとメスを同居させる。過度な交尾はメスの体力を消耗させるため管理が必要。

  • 温度回復後1～2週間で食欲が戻ったら開始
  • オスのケージにメスを入れる（縄張り争い防止）
  • 交尾確認後は速やかに隔離しメスに栄養補給
  • 複数回の場合は2～3日おきに同居
  • 攻撃性が見られたらすぐに隔離
💡 ボールパイソンは交尾確認後24時間で隔離が基本。
// PHASE 03
産卵・出産

卵胎生種（ガーターヘビ等）は出産、卵生種は産卵床が必要。産卵床を掘る行動が産卵直前のサイン。

  • 交尾から産卵まで3～8週間（種により異なる）
  • 産卵床素材：バーミキュライト・湿らせたスファグナムモス
  • 産卵後は速やかに卵を回収しインキュベーターへ
  • 卵の向きは変えない（マーキングして同じ向きを維持）
  • 白く張りがある→有精卵、萎んで透明→無精卵の可能性
💡 卵の凹みはすぐ無精卵ではない。加湿で回復することも多い。
// PHASE 04
インキュベーション

温度・湿度管理が孵化率を左右する最重要工程。多くの種は温度依存型性別決定（TSD）を持つ。

  • レオパ：28～30℃、40～65日孵化
  • ボールパイソン：31～32℃、55～65日孵化
  • フトアゴ：28～30℃（32℃超でメス増加）、55～75日
  • コーンスネーク：27～29℃、60～70日孵化
  • 湿度80～90%維持。バーミキュライトに適量加水。
🌡 TSD：多くの種は高温→メス、低温→オスになる傾向がある。
// PHASE 05
ベビーの育成

幼体は環境変化に敏感。孵化後の初給餌・最初の脱皮完了までが最も重要な時期。

  • 孵化後24～48時間は臍が閉じるまで安静に
  • ヘビ系の初給餌は孵化後7～10日・最初の脱皮後が理想
  • 初給餌はSSピンクマウス・またはSSコオロギから
  • 成体よりやや高い温度（30～32℃）と高湿度を維持
  • 同腹でも個別管理推奨（共食い・ストレスリスク）
⚠ 幼体の販売・譲渡は動物取扱業登録が必要な場合あり。要確認。
// REFERENCE DATA
主要種の繁殖データ

各種の繁殖基本データの参考指標。個体差・環境差があるため目安として活用。

  • レオパ：年4～8クラッチ（2卵/回）
  • ボールパイソン：1クラッチ4～12卵
  • フトアゴ：1クラッチ15～35卵
  • コーンスネーク：1クラッチ10～30卵
  • エボシカメレオン：1クラッチ30～85卵（孵化180～240日）
  • リクガメ：1クラッチ2～15卵（孵化60～300日）
💡 モルフ掛け合わせにはメンデルの法則と爬虫類遺伝学の知識が必要。

// SEASONAL CALENDAR

季節の飼育カレンダー

月別の飼育管理ポイント。季節の変化に合わせて適切にケアしよう。

1-2
// 冬
厳冬期の管理
  • 保温器具の故障に注意。バックアップヒーター準備推奨
  • 温帯種のブルメーション（擬似冬眠）を邪魔しない
  • 乾燥対策：湿度が下がりやすいため霧吹き頻度を上げる
  • 消費電力が増えるため電気代を計算し直す
  • 給餌量は活性に合わせて減らしてOK
3-4
// 春
繁殖スタート・温度上昇
  • クーリング明け個体の温度を徐々に元に戻す
  • 発情したオスの攻撃性が高まる種に注意
  • ペアリング開始のタイミング。体重を確認してから
  • UVBランプの交換チェック（年度初めに交換推奨）
  • 春に食欲が急増する種が多い。給餌量を戻す
5-6
// 初夏
産卵期・梅雨対策
  • 産卵床の準備。ケージ内に湿った産卵スペースを確保
  • 梅雨時期の過湿度に注意（砂漠種は除湿が必要な場合も）
  • インキュベーターの温度設定を見直す
  • エアコン管理の開始を検討（特に高温弱種）
⚠ カメレオン・ニューカレドニア産ヤモリは25℃超で危険。エアコン必須。
7-8
// 真夏
熱中症対策が最重要
  • 室温が30℃を超えるとほぼすべての種に悪影響
  • エアコン24時間稼働推奨（停電対策も忘れずに）
  • 水入れの水は1日2回交換。細菌繁殖しやすい
  • 停電対策：保冷剤・ポータブル電源の準備
⚠ 真夏の直射日光・車内放置は即死リスク。絶対にNG。
9-10
// 秋
クーリング準備・食欲旺盛期
  • 繁殖予定種のクーリング開始（体重確認後）
  • この時期は多くの種で食欲が最高潮。栄養補給のチャンス
  • 温帯種の越冬準備（室内越冬 or 冬眠）の判断
  • ヒーター類の動作確認・消耗品の買い置き
11-12
// 初冬
越冬体制・保温強化
  • 保温球・パネルヒーターを増設し温度勾配を維持
  • サーモスタットの設定を冬季モードに変更
  • 乾燥する季節。ウェットシェルターの水を毎日補充
  • 拒食が増える時期。体重管理を徹底
