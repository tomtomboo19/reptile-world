// DANGEROUS SPECIES

危険種・毒ヘビと応急処置

飼育・遭遇時に知っておくべき危険な爬虫類。万が一に備えた知識を。

⚠ このページは教育・情報提供を目的としています。毒ヘビ・危険種に咬まれた場合は直ちに救急車（119番）を呼び、自己処置を最小限にして病院へ搬送してください。
DEADLY / 致死的
ガブーンバイパー
Bitis gabonica

アフリカ産の大型毒ヘビ。世界最長の牙（5cm超）と強力な出血毒を持つ。飼育には都道府県知事の許可が必要（特定動物）。

// 応急処置絶対に安静にし動かない。患部を心臓より低く保つ。切開・吸引・縛るのは禁忌。119番通報後、咬んだヘビの種類を記録して病院へ。抗毒素血清が必要。
DEADLY / 致死的
コモドドラゴン
Varanus komodoensis

世界最大のトカゲ。毒腺と口腔内細菌により咬傷から重篤な感染症・毒素中毒を引き起こす。飼育は特定動物指定のため許可制。

// 応急処置深い咬傷を受けた場合は出血を抑えながら直ちに救急搬送。細菌感染予防のため病院での抗生物質投与が必須。
VENOMOUS / 強毒
タイパン
Oxyuranus scutellatus

オーストラリア産の世界最強クラスの神経毒を持つヘビ。一回の咬傷で複数人を殺傷できる量の毒を持つ。飼育は事実上不可。

// 応急処置神経毒のため症状発現が早い。意識・呼吸の確認を続けながら即搬送。圧迫固定法（バンデージで患部から心臓方向に巻く）が有効。
VENOMOUS / 強毒
ラッセルクサリヘビ
Daboia russelii

インド～東南アジア産。強力な出血毒・腎毒性を持ち、世界で最も多くの人間を死亡させるヘビのひとつ。特定動物指定。

// 応急処置出血毒のため内出血・腎不全が危険。患部を動かさず即搬送。縛るのは禁忌。
CAUTION / 要注意
ダイヤモンドバックガラガラヘビ
Crotalus atrox

北米産のガラガラヘビ。出血毒・壊死毒を持つ。尾の「ラトル」音が警戒サイン。特定動物指定のため飼育には許可が必要。

// 応急処置安静を保ち119番へ。腫れが広がるため指輪・時計等は早めに外す。抗毒素血清の投与が治療の基本。
CAUTION / 要注意
マムシ（日本産）
Gloydius blomhoffii

日本全国に分布する唯一の危険な毒ヘビ。山道・草むらで遭遇リスクあり。重篤な組織壊死を起こす。

// 応急処置安静・保温して119番へ。咬まれた部位を心臓より低くする。吸引・切開は感染リスクで推奨されない。病院でのセロバマ投与が標準治療。
