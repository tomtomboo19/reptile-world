Varanus prasinus

エメラルドツリーモニター

モニター 上級者向け 上級者向け樹上棲鮮やかなグリーン
🦕
75～100cm全長
10～15年寿命
30～35℃温度目安
// 基本情報

エメラルドグリーンの体色が非常に美しい樹上性モニター。飼育は難しく上級者向け。

// 生息地

ニューギニア島の熱帯雨林

// 食性・餌

コオロギ・ゴキブリ・小型トカゲ

// 飼育のコツ

高湿度（70～80%）と高いバスキングスポット必須。ストレスに敏感。
