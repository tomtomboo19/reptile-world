Agrionemys horsfieldii

ロシアリクガメ

リクガメ 初心者向け 初心者向けリクガメ入門種丈夫
🐢
15～20cm全長
40～50年寿命
28～35℃温度目安
// 基本情報

リクガメ入門種として最も人気。小型で丈夫。冬眠させることもできる。

// 生息地

中央アジアの乾燥草原・砂礫地

// 食性・餌

葉物野菜・野草・ティモシー

// 飼育のコツ

高温多湿に弱い。夏は涼しい日陰を。初心者の冬眠は室内越冬が安全。
